# Usage

<!-- usage -->

```sh-session
$ npm install -g toast
$ toast COMMAND
running command...
$ toast (-v|--version|version)
toast/0.0.1 darwin-x64 node-v11.15.0
$ toast --help [COMMAND]
USAGE
  $ toast COMMAND
...
```
